코드 생성 프롬프트

**목표:**
사용자가 노드와 경로를 설정하고, 데이터 흐름 애니메이션을 GIF 파일로 내보낼 수 있는 간단한 웹 기반 툴을 만들고 싶습니다.

**핵심 기능:**
1.  **캔버스:** HTML `<canvas>` 요소를 사용해 다이어그램을 그립니다.
2.  **노드:** 사용자가 클릭하여 캔버스에 원형 노드를 추가할 수 있어야 합니다.
3.  **경로:** 최소 2개 이상의 노드가 있을 때, 사용자가 두 노드를 순서대로 클릭하면 두 노드를 잇는 경로(선)가 생성됩니다.
4.  **애니메이션:**
    - 경로를 따라 작은 원(데이터 흐름)이 이동하는 애니메이션을 구현합니다.
    - 애니메이션은 마지막 노드에 도착하면 다시 시작 노드로 돌아와 반복(루프)됩니다.
5.  **GIF 내보내기:**
    - "GIF로 저장" 버튼을 누르면 현재 캔버스에 표시되는 애니메이션을 **gif.js** 라이브러리를 사용해 GIF 파일로 변환하여 다운로드할 수 있어야 합니다.
    - GIF는 약 3초 길이로, 30fps(초당 프레임)로 생성합니다.
6.  **UI:** 노드를 추가하고, 경로를 설정하고, GIF를 저장할 수 있는 간단한 버튼들을 캔버스 옆에 배치합니다.

**기술 스택:**
* HTML5
* CSS3
* Vanilla JavaScript
* gif.js 라이브러리 (CDN 링크를 사용해주세요: `https://cdn.jsdelivr.net/npm/gif.js@0.2.0/dist/gif.js`)

**코드 구조:**
- `index.html` (HTML 구조)
- `style.css` (CSS 스타일)
- `script.js` (모든 JavaScript 로직)

**요청:**
위 내용을 바탕으로 `index.html`, `style.css`, `script.js` 파일을 모두 작성해 주세요. 코드는 각 파일별로 분리해서 제공하고, 주석을 통해 주요 로직을 설명해주면 좋습니다.
